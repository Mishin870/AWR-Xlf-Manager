How to translate

  1. Search for page ‘XLF Tools’ and open it

  2. Choose target language

  3. Import source file (from scratch or append to existing translations)

  4. Click ‘Options’ and setup translation service (See How to get API Keys)

  5. Translate all strings by clicking on one of translation buttons

  6. Additionally you can click on ‘Translations’ button and edit translated strings

  7. To export translations click on ‘Export XLF’

Descriptions

Import XLF (Clear)

Clear translations table and import source strings into translations table

Import XLF (Merge)

Import only not existing source strings into translations table

Options

Setup translation service and api keys

Translations

Manually translate some strings

Export XLF

Export translated strings as new XLF translation file with name from ‘Base file name’ and suffix from ‘Target language’ field

For example:
Target language = ‘ru-RU’
Base file name = ‘test’

Exported file: test.ru-RU.xlf

Translate (ALL)

Translate all source strings (using translation service from ‘Options’ and target language from ‘Target language’)

Translate (Untranslated)

Translate source strings only if they have not been translated (using translation service from ‘Options’ and target language from ‘Target language’)

Translate (From file, Overwrite)

Upload early translated file to fill ‘Target’ fields in Translations table from this file

How to get API keys

Yandex

  1. Register Yandex account https://passport.yandex.com/registration

  2. Go to https://translate.yandex.com/developers/keys and click on ‘Create a new key’

  3. Enter some description for new API key and click ‘Create’

  4. Copy generated API key by clicking on ‘Copy’ button

  5. Go to Options page and paste new Yandex API Key

Azure

  1. Go to https://portal.azure.com and log in to your account

  2. Click on ‘Create new resource’

  3. Search for ‘Translator Text’

  4. Click on ‘Create’

  5. Fill all fields and click ‘Create’

  6. After deployment click on ‘Go to resource’

  7. Select ‘Keys’ tab and copy Key 1

  8. Go to Options page and paste new Azure API Key
